КТ №2 Тестирование функционала веб-приложения с применением техник тестирования

Используемое веб-приложение
https://the-internet.herokuapp.com/

Причины выбора:
- демонстрационный сайт для Selenium
- содержит окна и iframe
- стабильно работает

Реализованные тесты

 Работа с окнами (Windows)
- открытие нового окна
- переключение между окнами
- проверка заголовка
- возврат в основное окно

Работа с iframe
- переключение в iframe
- ввод текста
- возврат на основную страницу

Используемые технологии
- Python
- Selenium WebDriver
- PyTest
- ChromeDriver

Запуск проекта
```bash
pip install -r requirements.txt
pytest -s
```
